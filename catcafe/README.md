### *Actividad4 Front end en equipo*
This folder contains the website our team developed for **Activity 4**.

## Description:
The website has 3 tabs:
- *Our Café:* In this tab, there is information about the cat café, the menu and current offers
- *Our Cats:* In this tab, there is information about the cats and their adoption profiles
- *What is a Cat Café?:* In this tab, there is a brief explanation of what cat cafés are and their origin

## Features:
- **Nav Pills navigation:** The website is divided into 3 sections using Bootstrap Nav Pills
- **Dynamic menu:** The menu changes depending on the day of the week selected by the user
- **Image on hover:** Each menu item displays an image when the user hovers over it
- **Adopt section:** Cat information and image are hidden until the user clicks the button
- **Animations:** 
  - Menu content fades in with a text tracking animation each time a new day is selected
  - Images animate upward when hovered
  - Cat image rotates on its axis when hovered
